# NPRG030-Program

Zápočtový program pro předmět NPRG030. Matfyz - Programování 1, 2022/23 Zimní semestr

## Prerequisites

This program uses:
- `pynput`

`pip install -r requirements.txt`

## Controls

- left/right arrow
  - moves the block left or right
- down arrow
  - moves the block down
- up arrow
  - rotates the block
- esp
  - quit the game
